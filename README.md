# Sensor Temperature Analysis

## 📌 Project Overview
This project analyzes daily temperature readings from 4 weather stations using NumPy.

## 📊 Dataset
- 4 Weather Stations
- 5 Daily Readings Each
- Total Shape: (4, 5)

## ✅ Tasks Completed

### Task 1
- Printed shape of the dataset
- Calculated mean temperature per station (row-wise)

### Task 2
- Used Boolean mask to extract temperatures above 28°C
- Returned results as 1D array

### Task 3
- Normalized entire dataset to range [0,1]
- Rounded results to 2 decimal places

## 🛠 Tools Used
- Python
- NumPy
- Google Colab
